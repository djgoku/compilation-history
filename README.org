#+TITLE: Compilation History

This package provides automatic tracking of compilation history in Emacs.
It captures compilation commands, timing, results, and metadata to help
analyze build patterns and debug compilation issues.

* Features

- Automatic tracking of compilation history.
- Stores compilation data in an SQLite database.
- Provides a unique buffer name for each compilation history.

* Usage

To use this package, you need to initialize it first:

#+begin_src elisp
(compilation-history-init)
#+end_src

Then, you can enable the minor mode to automatically use the custom buffer naming scheme:

#+begin_src elisp
(compilation-history-mode 1)
#+end_src

You can disable it with:

#+begin_src elisp
(compilation-history-mode -1)
#+end_src

* Buffer Name

The buffer name for each compilation history is generated automatically
with the following format:

`*compilation-history-<identifier>==<path>__<command>*`

**Components:**

- `*...*`: The asterisks are an Emacs convention for special buffers that should not be saved to a file.
- `compilation-history-`: The prefix for all compilation history buffers.
- `<identifier>`: A timestamp in the format `YYYYMMDDTHHMMSSffffff`. This serves as the unique ID for the compilation.
- `<path>`:
  - If the compilation is run from the project root, this is the `<project-name>`.
  - If the compilation is run from a subdirectory, this is `<project-name>--<subdirectory-parts>`, where `<subdirectory-parts>` are the parts of the path from the project root, joined by `--`.
- `<command>`: The sanitized compile command.
  - Special characters are replaced with a hyphen.
  - Consecutive hyphens are collapsed to a single hyphen.
  - The command is truncated to a configurable length, with a default of 25 characters.

**Separators:**

- `==`: Separates the identifier and the path.
- `__`: Separates the path and the command.

**Examples:**

- Compilation at the project root (`my-project`):
  `*compilation-history-20250812T153000123456==my-project__make-all*`

- Compilation in a subdirectory (`/src/app`):
  `*compilation-history-20250812T153000123456==my-project--src--app__make-all*`
